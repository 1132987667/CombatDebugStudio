# 游戏功能介绍

## 1. 角色属性系统

### 1.1 基础属性列表

| 属性名称 | 编码 | 定义 | 数值范围 | 对游戏体验的影响 |
|---------|------|------|---------|----------------|
| 等级 | level | 角色的等级 | 1-99 | 影响角色基础属性值和技能解锁 |
| 名称 | name | 角色的名字 | - | 用于识别和区分不同角色 |
| 能量 | currentEnergy | 角色当前能量值 | 0-100 | 用于施放技能，影响技能释放频率，初始值为25 |
| 气血 | currentHp | 角色当前生命值 | 0-最大值 | 直接影响角色生存能力，为0时角色死亡 |
| 最小攻击-最大攻击 | minAttack/maxAttack | 角色攻击伤害范围 | 1-9999 | 直接影响伤害输出，范围越大伤害波动越大 |
| 防御 | defense | 角色抵抗伤害的能力 | 0-9999 | 减少受到的伤害，值越高减伤越多 |
| 速度 | speed | 角色行动顺序的决定因素 | 1-9999 | 速度越高，行动顺序越靠前，回合内行动次数可能增加 |
| 暴击率 | critRate | 攻击产生暴击的概率 | 0-100% | 提高暴击触发几率，增加伤害爆发能力，默认为10% |
| 暴击伤害 | critDamage | 暴击时的伤害倍率 | 100-500% | 暴击时造成的额外伤害，值越高暴击伤害越高，默认125% |
| 气血% | healthBonus | 气血加成百分比 | 0-500% | 提高角色气血上限，增强生存能力 |
| 攻击% | attackBonus | 攻击加成百分比 | 0-500% | 提高角色攻击力，增强伤害输出 |
| 防御% | defenseBonus | 防御加成百分比 | 0-500% | 提高角色防御力，增强生存能力 |
| 速度% | speedBonus | 速度加成百分比 | 0-500% | 提高角色速度，增强行动能力 |

### 1.2 能量系统详解

#### 1.2.1 能量获取与消耗
- **能量获取**：角色每回合结束时自动恢复25点能量
- **能量消耗**：施放技能需要消耗对应能量值
  - 小技能：固定消耗50点能量
  - 大技能（终极技能）：固定消耗150点能量
  - 被动技能：无能量消耗

#### 1.2.2 能量策略
- **能量管理**：合理规划技能释放顺序，确保关键技能在需要时能够释放
- **爆发时机**：积累足够能量后，可选择释放高伤害的大技能
- **持续输出**：通过小技能的循环使用，保持稳定的伤害输出

#### 1.2.3 能量上限与特性
- **能量上限**：角色能量值上限为100点
- **能量溢出**：当能量恢复后超过上限时，会自动保持为100点
- **初始能量**：角色初始能量值为25点，第一回合即可释放小技能

### 1.3 属性影响机制

- **属性相互关系**：某些属性会相互影响，例如高防御会降低受到的伤害，高速度会增加行动次数
- **属性上限**：部分属性存在理论上限，如暴击率最高为100%
- **属性成长**：角色等级提升时，基础属性会相应增长
- **属性加成**：通过装备、技能、Buff等方式可以临时或永久提升属性

## 2. 战斗计算公式

### 2.1 伤害计算公式

#### 基础伤害计算
```
基础伤害 = 随机值(最小攻击-最大攻击) × 各种伤害修正系数
```

#### 防御减免计算
```
伤害减免 = 目标防御值 × 0.01
最终伤害 = 基础伤害 × (1 - 伤害减免)
```

#### 暴击伤害计算
```
暴击伤害 = 最终伤害 × 暴击伤害倍率
```

### 2.2 命中与闪避计算

#### 命中概率计算
```
实际命中概率 = 攻击者命中率 - 目标闪避率
```

#### 闪避判定
```
是否闪避 = 随机数 < 目标闪避率
```

### 2.3 暴击概率计算
```
实际暴击概率 = 角色基础暴击率 + 技能/Buff加成
```

## 3. 游戏核心机制

### 3.1 基础玩法

- **回合制战斗**：游戏采用回合制战斗系统，角色按照速度顺序行动
- **技能释放**：角色拥有多种技能，消耗能量值释放
- **策略搭配**：玩家需要根据敌人特点和战场形势，合理搭配角色和技能
- **资源管理**：管理角色的能量和生命值，确保战斗持续能力

### 3.2 操作方式

- **角色选择**：在参战面板中选择需要出战的角色
- **技能释放**：在战斗界面中点击角色头像，然后选择要释放的技能
- **目标选择**：释放技能时选择目标敌人或友军
- **战斗控制**：可暂停、加速或重置战斗

### 3.3 胜利条件

- **全歼敌人**：击败所有敌方角色
- **目标达成**：完成特定任务目标（如保护特定角色存活）
- **时间限制**：在规定回合数内达成目标

### 3.4 失败判定规则

- **我方全灭**：所有我方角色生命值降至0
- **任务失败**：未能在规定条件下完成任务目标
- **时间耗尽**：超过规定回合数仍未达成胜利条件

| 生命回复 | 💚 | 每回合回复50点生命值 | 3回合 | 技能释放、装备效果 | 独立叠加，每层单独计算 |

#### 4.1.2 减益效果（Debuff）

| Buff名称 | 图标 | 效果描述 | 持续时间 | 获取途径 | 叠加规则 |
|---------|------|---------|---------|---------|--------|
| 中毒 | 🟢 | 每回合受到20点伤害 | 4回合 | 技能释放、敌人攻击 | 独立叠加，每层单独计算 |
| 眩晕 | ⭕ | 无法进行任何行动 | 1回合 | 技能释放、敌人攻击 | 不可叠加，刷新持续时间 |

| 暴击伤害降低 | 💥 | 暴击伤害降低30% | 2回合 | 技能释放、敌人攻击 | 刷新持续时间 |
| 重伤 | 💔 | 受到伤害增加20% | 3回合 | 技能释放、敌人攻击 | 刷新持续时间 |

### 4.2 Buff叠加规则

- **独立叠加**：每层Buff独立计算效果，如中毒效果每层单独造成伤害
- **刷新持续时间**：新Buff覆盖旧Buff并重置持续时间，如攻击力提升
- **限制层数**：达到最大层数后无法继续叠加，如某些特殊Buff最多叠加3层

### 4.3 Buff系统特性

- **可驱散性**：部分Buff可被驱散技能移除
- **免疫性**：某些角色或状态对特定Buff免疫
- **控制效果**：部分Debuff带有控制效果，如眩晕、沉默等
- **持续时间**：Buff通常有明确的持续回合数，回合结束后自动消失

## 5. 游戏功能模块

### 5.1 参战面板

- **入口位置**：游戏主界面左侧
- **核心作用**：管理出战角色，调整队伍配置
- **使用方法**：
  1. 勾选角色前方的复选框以启用/禁用角色
  2. 点击角色头像查看详细属性
  3. 通过拖拽调整角色出战顺序

### 5.2 战斗界面

- **入口位置**：游戏主界面中央
- **核心作用**：进行战斗操作，观察战斗过程
- **使用方法**：
  1. 点击角色头像选择要行动的角色
  2. 从技能列表中选择要释放的技能
  3. 点击目标敌人或友军确定技能目标
  4. 使用界面下方的控制按钮调整战斗速度或暂停

### 5.3 调试面板

- **入口位置**：游戏主界面右侧
- **核心作用**：实时监控角色属性变化，查看详细战斗数据
- **使用方法**：
  1. 选择要查看的角色
  2. 观察属性监控区域的实时数据变化
  3. 查看技能信息和Buff效果

### 5.4 战斗日志

- **入口位置**：游戏主界面底部
- **核心作用**：记录战斗过程中的所有事件和数据
- **使用方法**：
  1. 滚动查看详细战斗记录
  2. 分析伤害数据和Buff效果
  3. 排查战斗中的异常情况

### 5.5 场景管理

- **入口位置**：游戏主界面顶部菜单
- **核心作用**：切换不同战斗场景，调整战斗环境
- **使用方法**：
  1. 从场景列表中选择要进入的场景
  2. 查看场景描述和敌人配置
  3. 点击"进入场景"开始战斗

### 5.6 战斗规则设置

- **入口位置**：游戏主界面顶部菜单
- **核心作用**：调整战斗规则和参数，自定义游戏体验
- **使用方法**：
  1. 修改战斗参数如伤害倍率、暴击概率等
  2. 启用或禁用特定战斗机制
  3. 保存设置并应用到战斗中
